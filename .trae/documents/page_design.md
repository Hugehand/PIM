# 页面设计 - 个人信息管家

## 1. 设计系统
- **颜色**: 干净、专业的色调 (Slate/Zinc/Blue)。
- **排版**: 无衬线字体，易读。
- **布局**: 响应式侧边栏导航 (桌面端) 或底部导航 (移动端)。

## 2. 页面流程

### 2.1 首页 / 仪表盘 (Dashboard)
- **目标**: 快速访问常用操作。
- **元素**:
    - "快速生成" 卡片：选择最近使用的模板并复制。
    - "档案完善度" 状态。
    - "最近模板" 列表。

### 2.2 档案管理 (`/profile`)
- **布局**: 标签页或侧边栏分节 (基本信息, 教育经历, 工作经历, 家庭成员)。
- **基本信息标签**: 个人详情的表单字段。
- **教育/工作/家庭标签**: 带有 "添加", "编辑", "删除" 按钮的列表视图。
- **交互**: 实时保存或显式的 "保存" 按钮 (本地应用首选实时保存)。

### 2.3 模板库 (`/templates`)
- **列表视图**: 可用模板的网格或列表。
- **创建/编辑视图**:
    - **名称输入**。
    - **格式选择器**: 文本, JSON, 表格。
    - **编辑区域**: 定义模板结构的文本区域，带有变量插入助手（例如，点击 "插入姓名" 以添加 `{{basic.name}}`）。
    - **预览区域**: 使用虚拟数据或实际档案数据的实时预览。

### 2.4 生成器 (`/generate`)
- **步骤 1**: 选择模板。
- **步骤 2**: 预览输出。
- **步骤 3**: 操作按钮 (复制到剪贴板, 下载)。

## 3. UI 组件 (MVP)
- **输入框**: 标准文本, 日期, 选择输入框。
- **卡片**: 用于分组信息的容器。
- **按钮**: 主要 (复制, 保存), 次要 (取消, 编辑)。
- **模态框**: 用于添加/编辑多行项目 (教育, 工作)。
- **提示框 (Toast)**: 用于成功消息 ("已复制到剪贴板!")。
