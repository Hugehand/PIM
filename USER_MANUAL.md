# 个人信息管家 (PIM) - 用户操作手册

欢迎使用个人信息管家！本平台旨在帮助您一次性录入个人信息，并快速生成适用于各种场景（如简历、报名表、政审材料等）的格式化文本。

## 1. 快速开始

### 第一步：访问平台
请在浏览器中打开：[https://traesskebf2n.vercel.app/](https://traesskebf2n.vercel.app/)

### 第二步：录入基本信息
1. 点击左侧菜单的 **“基本信息”**。
2. 填写您的姓名、性别、民族、出生日期、政治面貌等信息。
3. **注意**：所有数据仅保存在您的本地浏览器中，我们不会上传到任何服务器，请放心填写。
4. 填写完毕后，系统会自动保存（或点击保存按钮）。

### 第三步：添加经历
1. 点击左侧菜单的 **“教育/工作经历”**。
2. 点击 **“添加经历”** 按钮。
3. 填写开始时间、结束时间。
4. 在“毕业院校及所学专业（工作单位及职务）”栏中，详细填写相关信息。
   - *示例：山东大学 计算机科学与技术专业*
   - *示例：某某科技有限公司 高级Java工程师*
5. 点击 **“保存”**。您可以添加多条经历。

### 第四步：添加家庭成员
1. 点击左侧菜单的 **“家庭成员”**。
2. 点击 **“添加成员”**。
3. 填写称谓（如父亲、母亲）、姓名、政治面貌、工作单位及职务。
4. 点击 **“保存”**。

## 2. 生成与导出数据

当您需要使用这些信息时：

1. 点击左侧菜单的 **“生成器”**。
2. 在 **“选择模板”** 下拉框中，选择您需要的格式：
   - **默认模板**：包含所有信息的综述。
   - **简历格式**：适合用于简历制作。
   - **政审表格式**：适合填写各类官方表格。
   - **纯文本**：适合复制到记事本。
   - **JSON**：适合开发人员或数据备份。
3. 在右侧的预览框中，您将看到生成好的文本。
4. 点击 **“复制”** 按钮，即可将内容粘贴到您的文档中。

## 3. 数据安全与隐私

- **本地存储**：本平台采用 LocalStorage 技术，所有数据均存储在您当前使用的浏览器中。
- **清除数据**：如果您在公共电脑上使用，请务必在使用完毕后，点击设置中的“清除所有数据”或手动清除浏览器缓存，以防个人信息泄露。

## 4. 常见问题 (FAQ)

**Q: 换了电脑或浏览器，数据还在吗？**
A: 不在。因为数据存储在本地，更换设备或浏览器后需要重新录入。建议您定期导出 JSON 格式的数据进行备份，在由于设备上可以导入（导入功能开发中）。

**Q: 为什么生成的日期格式不对？**
A: 请检查您在录入时选择的日期是否准确。系统会默认使用 `YYYY-MM-DD` 格式。

**Q: 遇到Bug怎么办？**
A: 请访问我们的 [GitHub 仓库](https://github.com/Hugehand/PIM) 提交 Issue 反馈。
