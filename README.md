# 产品需求文档 (PRD) - 个人信息管家

在线使用：https://traesskebf2n.vercel.app/

## 1. 产品概述
**名称**：个人信息桥 (暂定名)
**核心价值**：一个“个人主数据”管理工具，允许用户一次性录入信息，并生成适用于考试、申请和报名的各种格式。
**目标用户**：求职者（公务员、教师）、学生（研究生、留学）、专业人士（考证、职称评审）。

## 2. 核心功能 (MVP)

### 2.1 个人信息主档
用户需要以结构化的方式录入和管理个人数据。
- **基本信息**：姓名、性别、民族、出生日期、证件类型/号码、联系方式、邮箱、地址（支持现居住地/户籍地址等别名）。
- **教育经历**：学校、专业、学历、起止时间、学位。支持多条记录。
- **工作经历**：单位名称、岗位、起止时间、工作内容。支持多条记录。
- **家庭成员**：关系、姓名、单位、职务、联系方式。支持多条记录。
- **字段别名**：支持多名称识别字段（例如，“电话” vs “手机号”）。

### 2.2 模板驱动的格式转换
将主数据映射到特定的输出格式。
- **模板定义**：用户可以定义或选择模板。
- **格式类型**：
    - **纯文本**：带有特定分隔符的拼接字符串。
    - **键值对**：例如，“姓名：[值]”。
    - **JSON**：用于技术系统输入的结构化数据。
    - **表格/网格**：制表符分隔或 CSV，用于粘贴到 Excel/Word。
- **映射逻辑**：将“主数据字段”映射到“模板字段”。

### 2.3 一键复制/导出
- **预览**：在复制前查看生成的数据。
- **复制**：一键复制到剪贴板。
- **导出**：下载为 .txt, .json 或 .csv（MVP 阶段可选）。

## 3. 非功能性需求
- **隐私**：所有数据必须本地存储（LocalStorage/IndexedDB）。无需服务器端存储用户数据。
- **性能**：即时生成和复制。
- **易用性**：界面整洁简单。易于在档案编辑和生成之间切换。

## 4. 未来规划
- 浏览器插件自动填充表单。
- 云同步（可选，加密）。
- 社区模板分享。
